# 🔒 비밀 번역기

친구들과 함께 비밀 메시지를 암호화해서 공유하는 도구입니다!

## 기능

- 🔒 **암호 만들기**: 평문을 Base64로 암호화
- 🔓 **암호 해독하기**: 암호를 원래 메시지로 복호화
- 🌍 **온라인 배포**: GitHub Pages를 통해 언제 어디서나 접속 가능

## 사용 방법

1. [배포 페이지](https://joonhyuk-dotcom.github.io/secret-translator/)에 접속
2. 메시지를 입력
3. "🔒 암호 만들기" 버튼 클릭
4. 암호화된 메시지를 친구에게 공유
5. 친구가 암호화된 메시지를 입력 후 "🔓 암호 해독하기" 클릭

## 예시

- **원문**: "만나자"
- **암호**: "X_JUzSwmJQ==_Z"
